# Contributing to cuPQC

Thank you for your interest in contributing to the cuPQC examples!

## 1. Report bugs, request features, or share feedback

File an [issue](https://github.com/NVIDIA/cuPQC/issues) describing what you encountered or what you want to see changed. The NVIDIA team will evaluate and triage issues.

**Note:** At this time we do not accept code contributions. You are welcome to share feedback on existing implementations via [GitHub Issues](https://github.com/NVIDIA/cuPQC/issues).

## 2. Share your work built with cuPQC

Share your projects and use cases on [GitHub Discussions](https://github.com/NVIDIA/cuPQC/discussions) or ask cuPQC-related questions.

## Code of Conduct

This project follows the [Code of Conduct](CODE_OF_CONDUCT.md). By participating, you agree to abide by its terms.
